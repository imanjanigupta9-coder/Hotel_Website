## Customization Hotspots

### Change Carousel Speed
File: `static/js/carousel.js`
Line: 9
```javascript
this.autoRotateDelay = 4000; // milliseconds
```

### Add Testimonials
File: `app.py`
Section: TESTIMONIALS array
Add object with: quote, author, location, image, rating, badges

### Change Hero Images
File: `templates/index.html`
Lines: 312, 329
Replace image URLs

### Adjust Heights
File: `static/css/style.css`
Lines: 180-200
Modify height values in media queries
